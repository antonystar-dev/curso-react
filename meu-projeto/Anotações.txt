Vamos relembrar o React
1- Para instalar o react tenho q acessar o site https://create-react-app.dev/ e la encontro o comando 
de instalação, que seria: npx create-react-app my-app

2 -Quando instalei a primeira vez o React, tive que digitar cd meu-projeto para entrar no diretorio do projeto.
Para iniciar o react eu tenho q usar o npm start

3- A arvore do React conta com uns arquivos temporarios, praticamente so uso as pastas public e src, as demais são
 temporarias e quando reinstalo elas são geradas.

4- o projeto é setado no arquivo App.js, ele que é a pg principal que puxa os componentes.

5- A declaração da palavra Class no react será className, visto que Class é uma palavra reservada.

6- Dentro da pasta src é criado a pasta Components na qual tem a função de guardar a estrutura do site(arvore)
posso criar outras subpastas para melhor organização(forms, buttons....)

7- Posso exportar um componente dentro de outro componente. As props são valores passados para os componentes;
